### 1. **Missing robots.txt** 🔴 URGENT
**Problem:** No robots.txt file exists (404 error)
**Impact:** Search engines don't know what to crawl, may be crawling inefficiently
**Fix:**
```
Create: /robots.txt

User-agent: *
Allow: /
Sitemap: https://clearskyspeechutah.com/sitemap.xml

# Allow all major search engines
User-agent: Googlebot
Allow: /

User-agent: Bingbot
Allow: /

User-agent: Slurp
Allow: /

# Block AI scrapers if desired
User-agent: GPTBot
Disallow: /

User-agent: ChatGPT-User
Disallow: /

User-agent: CCBot
Disallow: /
```

### 2. **Missing XML Sitemap** 🔴 URGENT
**Problem:** No sitemap.xml exists
**Impact:** Search engines can't efficiently discover all your pages
**Fix:** Generate sitemap with all pages:
- Homepage
- 5 service pages
- 5 location pages
- About, Process, FAQ, Contact
- Blog posts

**Sitemap structure needed:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://clearskyspeechutah.com/</loc>
    <lastmod>2026-05-28</lastmod>
    <changefreq>monthly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://clearskyspeechutah.com/services.html</loc>
    <changefreq>monthly</changefreq>
    <priority>0.9</priority>
  </url>
  <!-- All other pages -->
</urlset>
```

## 🟡 HIGH PRIORITY ISSUES

### 4. **Location Pages Missing Schema Markup**
**Problem:** Location pages don't have LocalBusiness schema
**Impact:** Missing rich snippets in local search results
**Fix:** Add to each location page:

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Clear Sky Speech Therapy - Park City",
  "description": "Speech-language therapy services in Park City, Utah",
  "url": "https://clearskyspeechutah.com/locations/park-city-utah.html",
  "telephone": "+1-435-248-2135",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Park City",
    "addressRegion": "UT",
    "postalCode": "84060",
    "addressCountry": "US"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "40.6461",
    "longitude": "-111.4980"
  },
  "areaServed": {
    "@type": "City",
    "name": "Park City"
  },
  "priceRange": "$$",
  "image": "https://clearskyspeechutah.com/images/park-city-location.jpg"
}
</script>
```

### 5. **Service Pages Missing FAQ Schema**
**Problem:** FAQ sections don't have schema markup
**Impact:** Missing FAQ rich snippets in search
**Fix:** Add FAQPage schema to FAQ section on service pages

### 6. **Blog Posts Missing Article Schema**
**Problem:** Blog posts lack structured data
**Impact:** Reduced visibility in Google News, missing rich snippets
**Fix:** Add Article/BlogPosting schema to all blog posts

### 7. **No Breadcrumb Schema**
**Problem:** Site navigation not marked up
**Impact:** Missing breadcrumb rich snippets
**Fix:** Add BreadcrumbList schema to all non-homepage pages
